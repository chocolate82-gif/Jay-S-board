# DayFlow AI - Landing Page

Context-AI 기반 하루 루틴 자동 설계 서비스의 랜딩 페이지입니다.

## 기술 스택

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (애니메이션)
- **React Icons**

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 배포

GitHub Actions를 통해 자동으로 GitHub Pages에 배포됩니다.
